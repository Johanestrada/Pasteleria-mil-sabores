# Pastelería Mil Sabores


## Tecnologías utilizadas
- **HTML5** para la estructura del sitio.
- **CSS3** para el diseño y estilo visual.
- **JavaScript (Vanilla JS)** para la lógica y la interacción del usuario.

---

## Funcionalidades principales
### Para clientes
- Registro y autenticación de usuarios.
- Compra de productos online con carrito de compras.
- Aplicación de descuentos:
  - 50% para usuarios mayores de 50 años.
  - 10% con el código `FELICES50`.
  - Torta gratis en cumpleaños para correos @duoc.cl.
- Visualización de catálogo filtrado por categorías.
- Página “Nosotros” y sección de blogs.
- Formulario de contacto con validaciones.

### Para administrador
- Panel de administración para:
  - Listar, crear, editar y eliminar productos.
  - Gestionar usuarios.
- Visualización de pedidos y confirmaciones de compra.
